# tharinex_internship

Data Cleaning and Visualization Project

Introduction:
The purpose of this project was to clean and prepare a dataset for analysis using Python. Raw data often contains missing values and duplicate records that can affect the accuracy of analysis.
Dataset Description:
The dataset contains student academic scores, including mathematics and science scores.
Data Cleaning Process:
Checked for missing values, replaced missing numerical values with averages, removed duplicate records, and verified data consistency.
Tools Used:
Python, Pandas, and CSV files. Results Missing values were handled, duplicate records were removed, and the dataset became more reliable for analysis. 
Conclusion:
This project demonstrates the importance of data cleaning in preparing data for analysis and visualization.
